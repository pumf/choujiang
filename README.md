# 🎉 年会抽奖系统

一个基于 HTML + CSS + JavaScript 的年会抽奖系统，无需后台，数据存储在浏览器本地。

![年会抽奖系统](https://img.shields.io/badge/HTML5-CSS3-JS-yellowgreen)
![License](https://img.shields.io/badge/License-MIT-blue)
![Version](https://img.shields.io/badge/Version-1.0.0-brightgreen)

## ✨ 功能特性

### 🎰 抽奖功能
- **3D球形动画** - 点击开始抽奖后，所有未中奖人员聚拢形成3D球体，绕Y轴旋转
- **手动停止** - 随时点击「停止抽奖」结束抽奖，显示中奖名单
- **烟花特效** - 中奖后播放绚丽的烟花庆祝动画
- **取消抽奖** - 支持取消当前抽奖过程

### 👥 人员管理
- **添加人员** - 支持批量添加（每行一个姓名）
- **导入导出** - 支持 txt、csv、json 格式导入导出
- **数据持久化** - 所有数据保存在浏览器 LocalStorage

### 🏆 奖项设置
- **奖项管理** - 添加/删除奖项，设置名称、描述、人数、颜色
- **自动记录** - 抽取后自动记录中奖人
- **状态显示** - 实时显示奖项状态（待抽取/已抽取）

### 🎨 界面设置
- **自定义标题** - 设置年会抽奖标题
- **背景选择** - 4种预设渐变背景 + 本地图片上传
- **全屏模式** - 一键切换全屏/退出全屏

### 💾 数据管理
- **导出配置** - 导出全部数据为 JSON 文件
- **导入配置** - 导入备份的配置数据
- **清空数据** - 一键清空所有数据

## 🚀 快速开始

### 方式一：直接打开
```bash
# 直接在浏览器中打开 index.html 文件
open index.html
```

### 方式二：本地服务器
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000`

## 📖 使用说明

### 1. 添加奖项
1. 点击左下角「⚙️」打开设置面板
2. 在「奖项管理」区域填写奖项信息
3. 点击「添加奖项」

### 2. 添加人员
1. 点击左下角「⚙️」打开设置面板
2. 在「人员管理」区域输入姓名（每行一个）
3. 点击「添加人员」

### 3. 开始抽奖
1. （可选）点击左侧「奖项列表」选择要抽取的奖项
2. 点击中间的「🎰 开始抽奖」按钮
3. 等待球体聚拢并旋转
4. 点击「🛑 停止抽奖」结束抽奖
5. 查看中奖结果

### 4. 全屏模式
点击右下角「⛶」按钮切换全屏模式
